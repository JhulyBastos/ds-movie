## ⚙️ Instalação:

1. **Clone o repositório:**

```bash
$ git clone https://github.com/JhulyBastos/ds-movie.git
```

2. **Instale as dependências do frontend:**

   ```bash
   npm install
   ```

## 💻 Execução

Em seu terminal, execute o comando:

```bash
npm run dev
```

A aplicação irá rodar no endereço `http://localhost:3000/`
